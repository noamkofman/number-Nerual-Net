# Digit Neural Network from Scratch
This project was made with numpy and random as the only outside libraries. Using a MNIST datset of handrawn digits, 
their labels, and their pixel values I then run a dot product formula using 784 weights (28 pixels x 28 pixels = 784) 
each iteration of the loop it corrects itself by shifting weights based upon how much each pixel affects its decision, its error rate, and learning rate ( lr )
